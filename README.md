# Projeto-Agenda
Projeto de site de agenda com sistema de agendamento online, desenvolvido com Node.js, Express, MongoDB e Mongoose. Utiliza EJS para renderização dinâmica das páginas e npm para gerenciamento de dependências. Permite cadastro de usuários, autenticação, garantindo organização, segurança e controle eficiente dos horários.

🛠 Tecnologias Utilizadas

🔹 Backend
Node.js – Ambiente de execução JavaScript no servidor.
npm – Gerenciador de pacotes para instalação de dependências.
Express.js – Framework para criação do servidor e rotas.
Mongoose – ODM para modelagem e manipulação de dados no MongoDB.

🔹 Banco de Dados
MongoDB – Banco de dados NoSQL orientado a documentos.

🔹 Frontend
EJS – Template engine para renderização dinâmica das páginas.
HTML5
CSS3
